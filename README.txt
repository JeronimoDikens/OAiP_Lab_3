Лабораторная работа №6
Указатели. Динамическое распределение памяти
Цель работы: изучить операции работы с указателями и научиться использовать функции динамического выделения памяти для одномерных массивов

Задание: В некоторых видах спортивных состязаний выступление каждого спортсмена независимо оценивается несколькими судьями, затем из всей совокупности оценок удаляются наиболее высокая и наиболее низкая, а для оставшихся оценок вычисляется среднее арифметическое, которое и идет в зачет спортсмену. Если наиболее высокую оценку выставиои несколько судей, то из совокупности оценок удаляется только дна такая оценка; аналогично поступают с наиболее низкими оценками. 
 Даны натуральное число n, действительные положительные числа a1, a2, ... an(>=3). Считая, что числа a1,..., an  - это оценки, выставленные судьями одному из участников соревнований, определить оценку, которая пойдет в зачет этому спортсмену.
